# Quick Start Guide - Testing & CI/CD

## 🚀 5 Menit Setup

### Step 1: Install Dependencies
```bash
npm install -D vitest @testing-library/react @testing-library/jest-dom @vitest/ui @vitest/coverage-v8
```

### Step 2: Run Tests
```bash
npm test
```

### Step 3: Generate Coverage Report
```bash
npm run test:coverage
```

### Step 4: View Coverage in Browser
```bash
# Windows
start coverage/index.html

# macOS
open coverage/index.html

# Linux
xdg-open coverage/index.html
```

## 🔄 GitHub Actions Setup

### 1. Push to GitHub
Repository harus sudah di GitHub

```bash
git add .
git commit -m "Add testing & CI/CD setup"
git push origin main
```

### 2. GitHub Actions Auto-runs
Workflow `.github/workflows/test.yml` akan automatically execute:
- ✅ Setiap push ke main/develop
- ✅ Setiap Pull Request

### 3. Check Results
Buka repo di GitHub:
- Click "Actions" tab
- Lihat workflow run status
- Click run untuk detail

### 4. PR Comments
Workflow auto-comments coverage metrics di PR:

```
## 📊 Test Coverage Report

| Metric | Coverage |
|--------|----------|
| Lines | 100% |
| Statements | 100% |
| Functions | 100% |
| Branches | 100% |

✅ All tests passed!
```

## 📈 Codecov Integration

### 1. Connect Codecov (One-time)
1. Visit https://codecov.io
2. Sign in dengan GitHub
3. Authorize
4. Select repository
5. Done! ✓ (Token auto-generated)

### 2. After Each Test Run
Coverage automatically uploaded ke Codecov

### 3. View Reports
Visit: `https://codecov.io/gh/YOUR_USERNAME/ci-cd`

## 🛠️ Troubleshooting

### "npm not found"
**Solution**: Install Node.js from https://nodejs.org/

### "Tests not running"
**Solution**: 
```bash
npm install -D vitest
npm test
```

### "Coverage not generated"
**Solution**:
```bash
npm install -D @vitest/coverage-v8
npm run test:coverage
```

### "GitHub Actions not working"
**Solution**:
1. Check `.github/workflows/test.yml` exists
2. Commit & push file
3. Go to repo → Actions tab
4. Click workflow to see error
